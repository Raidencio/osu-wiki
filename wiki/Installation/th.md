# Installation (ไทย)

## วิธีการติดตั้ง

### คอมพิวเตอร์

#### osu!

##### Windows

###### สเปคขั้นต่ำ
- CPU: อะไรก็ได้ 1 GHz ขึ้นไป
- RAM: 512 MB ขึ้นไป
- [Microsoft .NET Framework (3.5+)](https://www.microsoft.com/th-th/download/details.aspx?id=48130)

###### วิธีการติดตั้ง
- ดาวน์โหลดโปรแกรมติดตั้ง [osu!](https://osu.ppy.sh/home/download)
- เปิดไฟล์ที่ดาวน์โหลดเเละทำตามคำสั่งเพื่อติดตั้ง osu!
- ไปที่หน้าจอเดสก์ทอปหรื่อสตาร์ทเมนูเเละกดไอคอน osu! เพ่อทีจะเปิดเกม
- จอเมนูหลักจะแสดงขึ้นมา ในหน้าต่างเกมจะมีตัวเลื่อกสองอย่างให้ล็อกอินบัญชี osu! (ถ้าต้องสร้างบัญชีกดที่ปุ่ม'สร้างบัญชีใหม่') หรื่อเล่นออฟไลน์
  - คุณจะสามารถบันทึกคะเเนนเเละมีตำเเหน่งอันดับโลก
- เล่นให้สนุกนะครับ ^^
