# Raccourcis clavier 

Pour accéder à cette fenêtre de dialogue, ouvrez les [Options](/wiki/Options), tapez dans  `change` puis cliquez sur `Change keyboard bindings`.

Ici, vous pouvez assigner de nouveaux raccourcis clavier avec les actions listées.

**Note :** le client osu! ne vous notifiera pas qu'une touche est utilisée plusieures fois (cela inclut les touches conflictuelles)!

## Mode standard d'osu! 

| Nom | Par défaut |
| :-- | :-- |
| Clic Gauche | `Z` |
| Clic Droit | `X` |
| Fumée | `C` |

| Nom | Par défaut |
| :-- | :-- |
| Bouger vers la gauche | `Left Arrow` |
| Bouger vers la droite | `Right Arrow` |
| Dash! (2x la vitesse de déplacement) | `Left Shift` |

## Mode osu!Mania

| Nom | Par défaut |
| :-- | :-- |
| Augmente la vitesse | `F3` |
| Diminue la vitesse | `F4` |

**Notes:**

- Vous pouvez aussi utiliser `Ctrl` + `+` pour augmenter la vitesse 
- Vous pouvez aussi utiliser `Ctrl` + `-` pour diminuer la vitesse 

## En jeux

| Nom | Par défaut |
| :-- | :-- |
| Mettre le jeu en pause | `Echap` |
| Passer la Cutscene | `Space` |
| Cacher/Montrer le tableau de score | `Tab` |
| Augmente l'offset local de la chanson | `+` (plus) |
| Augmente l'offset local de la chanson | `-` (moins) |
| Réessayer rapidement (a maintenir brièvement | `~` (tilde) |

**Notes:**

- Si vous changez la touche pour `Mettre en pause le jeu`, `Echap` peut toujours être utilisé pour mettre en pause le jeux.
- `Réessayer rapidement` demande au joueur de maintenir la touche pendant environ 2 secondes.

  - Pour désactiver la boss key, cliquez sur l'icône d'osu! dans la barre des tâches et le client osu! s'ouvrira.

**Notes:**

- Par défaut, ces raccourcis claviers sont positionnés de la même façon que les icônes des mods.
- Vous devrez peut être presser la touche deux fois pour accéder aux mods cachés en dessous d’eux.
  - exemple : Vous devez appuyer sur `S` deux fois pour accéder à Perfect.

## Remettre à zéro tous les raccourcis

**Attention:** ce bouton n'enverra **pas** de confirmation pour être sûr que vous voulez vraiment le faire !

Ce bouton remettra à zéro tous les raccourcis aux valeurs par défaut listées ci-dessus.

## Futilités

**Pour la sélection de mod:**

- Les mods xK en mania ne sont pas listés ici.
  - Vous pouvez appuyer sur les touches `3` à `9` pour activer les mods xK.
- Tous les icônes de mods n’ont pas de raccourci clavier, comme :
  - 1K
  - 2K
  - Random
  - Co-op
  - Target Practice

### Historique

- Les icônes des mods 1K et 2K quand ils avaient un raccourci clavier, `1` et `2` respectivement.
  - Ces raccourcis ont été supprimés car ils entraient en conflit avec les touches de navigation.
    (pressez `1` désélectionne l’icône du mod et  `2` fait sortir du menu des mods).
