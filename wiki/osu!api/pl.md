# osu!api

osu! posiada publiczne [API](https://en.wikipedia.org/wiki/API), którego używać mogą zewnętrzne usługi. W przeszłości miały do niego dostęp tylko upoważnione osoby, ale obecnie każdy użytkownik osu! może z niego korzystać.

Dokumentacja API znajduje się na [oficjalnej wiki osu-api](https://github.com/ppy/osu-api/wiki). Obecna wersja API korzysta z prywatnych kluczy do autoryzacji żądań. Możesz ubiegać się o klucz API na [stronie rejestracji aplikacji](https://osu.ppy.sh/p/api) osu!.

Miej na uwadze, że dokumentacja zawarta na wiki osu!api nie jest w pełni kompletna czy wyczerpująca, a samo API nie jest obecnie rozwijane. Nowa wersja API jest w trakcie powstawania na [repozytorium osu-web](https://github.com/ppy/osu-web).
